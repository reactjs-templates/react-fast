# 根据 Vue-cli 改成的适合 React 的生成项目

> A React.js project

## Build Setup

``` bash

# install dependencies
npm install

# serve with hot reload at localhost:8080, default port 8080.
# you can edit config/index.js update it.
npm run dev

# build for production with minification
npm run build

```

## 包含的内容

- React.js
- Redux
- Redux-thunk
- React-Router
- hot config
